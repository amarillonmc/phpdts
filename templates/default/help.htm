■FAQ 常见问题

下面以FAQ的形式将ACFUN大逃杀最基本的游戏概念作简要的阐述。
问：ACFUN大逃杀怎么玩？
答：ACFUN大逃杀是一个网页游戏，基本操作跟浏览其他网页是一样的，用鼠标选择指令并发送就可以进行游戏了。
至于更深入的知识，请大略把整份帮助文档浏览一遍，
第一次看可能很多地方一知半解，不用急着把它背起来，
只要头脑中有游戏概念，玩的时候遇到不了解的情况就查询帮助，
两三盘游戏之后差不多就会明白游戏流程了
答：如果你尚未注册，需要先点击屏幕上方、标题下方的 >>账号注册 ，并输入想要的用户名和密码来创建账号。
如果已经注册过，在首页登陆以后，单击按钮，或者屏幕上方的 >>进入游戏 链接皆可。
问：怎么移动？怎么捡东西？怎么找敌人？
答：在下拉框点选目的地，就可以移动到该地点。
点击则会在原地进行探索。
移动和探索都会消耗一定体力，如果自己处于异常状态还可能损失一定生命，而且可能发现物品、遇到敌人、队友、尸体、陷阱或者意外事件。
问：怎样杀人？怎样会死？
答：或者之后，有一定概率遭遇敌人。
如果对方发现了你，那么强制进入战斗；
如果对方没有发现你，那么你可以酌情选择或者。
战斗时依据双方攻击、防御以及熟练度，会互相造成损伤。
战斗中如果对方生命减为0，那么对方死亡，你可以在尸体上掠夺一件物品或者拿走金钱。
如果游戏没有进入连斗状态，以后还有可能再遇到同一个人的尸体，还有机会再拿物品，当然其他玩家也可能捷足先登。
如果你自己生命减为0，那么你将死去，在同一局游戏中无法再进行任何活动。
问：钱是干什么用的？怎么赚钱？怎么买东西？能不能卖东西？
答：钱可以在商店购买装备或者道具。
商店位于“无月之影”、“光坂高校”或者“花菱商厦”，在这些区域点击即可进入商店页面。
进入游戏时会自动给每个玩家发一定数量的金钱，NPC身上也有大量金钱，杀死玩家或者NPC就能拿到他身上携带的金钱。
除此之外，有些随机事件也会导致你获得金钱。目前没有别的更多获得金钱的方式，自然也不能卖东西。
另外，参与游戏或达成成就也可以获得“切糕”作为奖励。
“切糕”是一种账户货币，可以用来在 >>当前幸存 界面为游戏中的玩家购买补给
或在 >>幻想世界 页面提交原创合成、道具或NPC
问：能组队吗？怎么组队？怎样加入队伍？怎么给队友东西？给队友钱？
答：在游戏进入“连斗”阶段之前都可以组队，进入“连斗”之后所有队伍自动解散并且无法再组队，此时玩家只能自相残杀。
想创立队伍请点击，相应地还能和
创立队伍请直接输入你想用的队伍名称以及想用的密码，如果创立成功会消耗一定体力。
加入别人的队伍则输入对方的队名密码，加入成功也会消耗一定体力。
成为队友之后，只要可见度良好，队友之间都不会互相攻击。
遇到队友时，可以选择将道具送给队友，但是钱不能给队友。
起雾或者浓雾天气，可见度差，就算是队友也会误伤。
问：体力耗尽了！生命值快归零了！怎样回复体力和生命？
答：回复体力和生命有三种办法：吃补给品、睡眠/治疗/静养，以及等级提升。
游戏开局会发面包和矿泉水，分别能回复生命和体力。游戏各地区地面上也散布有大量的补给品，不过可能带毒，服用后反而损失生命。
如果补给耗尽，点击指令并提交可以随时间回复体力，选指令则是回复生命，在“精灵中心”以及“对天使用作战本部”还可以选择指令，同时回复体力和生命。
点击“提交”能看到回复量，在回复时，无法进行其他行动；点击“返回”并提交则返回作战状态。
此外，等级提升时也能回复少量体力和生命值。
问：受伤了怎么办？进入异常状态了怎么办？
答：受伤跟异常状态都不会自动解除。
受伤时点击界面人形上的，便可以消耗体力包扎对应的受伤部位。
如果进入异常状态，有三种办法解除：
一是只有寻找或购买相应的药剂并服用才能解除；
二是到“精灵中心”以及“对天使用作战本部”通过“静养”指令解除。如果你是某个特定称号，也可以直接点击异常状态本身将其解除。
三是点击，并点击“治疗”，即可消耗1点技能点解除所有异常状态。

2.游戏流程

【胜利条件】：
“ACFUN大逃杀”是玩家被迫互相残杀以争取唯一的生存名额的游戏。
因此，简要地说，杀光其他所有玩家，最后独存的那个人，就是本局游戏的胜利者，将迎来“最后幸存”结局。
此外，如果玩家设法杀死位于无月之影的BOSS，并使用从BOSS身上拾取的特殊道具，也可以带领剩余玩家逃离“ACFUN大逃杀”的世界，迎来“锁定解除”结局。
此外，也存在别的游戏结局，期待玩家可以找到。

【游戏路线】：
进入游戏 →→ 通过战斗提升等级和熟练 →→ 杀死NPC和玩家存钱买补给、武器、强化道具 →→ 连斗以后与其他玩家决斗并获胜 →→ 『最后幸存』
进入游戏 →→ 选择合适的姿势和策略 →→ 回避战斗并等待其他玩家自相残杀或被NPC杀死 →→ 躲藏在安全地点等待高等级玩家因禁区停留、踩高伤陷阱、撞BOSS或者其他RP死法而死亡 →→ 『最后幸存』
进入游戏 →→ 通过战斗提升等级和熟练 →→ 杀死NPC和玩家存钱买补给、武器、强化道具 →→ 用移动PC解除禁区 →→ 杀死位于无月之影的BOSS NPC →→ 『锁定解除』


3.禁区和连斗

ACFUN大逃杀地图中共有35个区域。
根据服务器的设定，每隔一定的时间，就会有一些地区变为禁区。
在禁区增加的那一刻，如果有人滞留在禁区，并且姿态没有调整为“躲避姿态”，那么这个人将会被强行杀死。

如果找到道具“移动PC”，玩家可以解除禁区。
从禁区解除的时刻起，一直到下一次增加禁区的时刻为止，玩家可以任意访问被禁止的区域。
如玩家想挑战游戏BOSS，也必须先解除禁区，才能到无月之影与BOSS决战。

当禁区数量增加到一定程度，或者在游戏中死亡的玩家和NPC总数超过$deathlimit人时，都将使游戏进入“连斗阶段”。
可以在游戏主页查看进入连斗阶段的条件。
进入连斗以后，游戏停止激活，所有尸体消失且无法再摸到尸体，所有队伍解散，躲避姿态也无法回避禁区，剩下的人必须决一死战。
同时禁区还会继续增加，如果全部地区都变为禁区，那么所有的参与者都会死亡，这局游戏将没有胜利者。

此外，在游戏进入连斗以后，如果玩家杀死了杏仁豆腐并使用掉落的“杏仁豆腐的ID卡”，则会使游戏进入“死斗模式”。
进入死斗模式以后，玩家将不会再遇到任何NPC，不过，如果想要快速消灭剩下的玩家，死斗模式不失为一种很好的选择。

5.指令说明

『移动』 消耗15点体力，离开现在的区域而移动到别的区域。
『探索』 消耗15点体力，探索现在的区域。
移动和探索都可能遇到敌人、队友、尸体，遇上突发事件，或者发现物品。
『道具使用』 根据道具的类型使用或者装备所选的道具。可拥有的道具上限是6个。

“道具”：
『道具合成』 将数个不同的道具，合成为一个新道具。常见的合成公式请参考下文
『道具丢弃』 将包裹里的道具丢掉。丢弃的道具不会消失，将被放置在当前的地区，自己或别人还能再次找到。
『整理包裹』 可以将完全相同的道具合并在一起，节约包裹空间。注意，将有毒的食物和无毒的食物合并，会使全部食物带毒。
『卸下装备』 将身上的武器或者防具卸下放置到包裹里。


如果你装备了「背包」属性的物品：
『存放道具』 将道具存入背包，存入的最大值为背包所示的效果。
『取出道具』 将道具从背包内取出。
『背包信息』 查看背包中的物品。

“恢复”：
『睡眠』 根据睡眠的时间和体力上限，按一定比例恢复体力。
『治疗』 根据睡眠的时间和生命上限，按一定比例恢复生命，但是不会治疗伤口或解除异常状态。伤口治疗在“特殊”功能内。
『静养』 同时恢复生命和体力，同时有一定概率解除异常状态，静养时间越长，解除概率越高。只能在精灵中心和对天使用作战本部内进行。

“特殊”：
『商店』 在无月之影、光坂高校或者花菱商厦时，可以进入商店购买道具。购买前请确认包裹里有空位放置道具，否则会要求丢弃一个道具。
『基础姿态』 更改自己的行动姿态，各种行动姿态的详情见下文。
『应战策略』 更改自己的应战策略，各种应战策略的详情见下文。
『留言变更』 修改自己的座右铭，杀人留言和遗言。
『武器带电』 给装备着的武器附加“电击”属性，只有特性为“锡安成员”时才具有此功能。需要消耗1个电池/探测器电池 。
『查毒』 检查饮食中是否有毒，只有称号为“黑衣组织”时才具有此功能。
『武器淬毒』 给装备着的武器附加“带毒”属性，只有称号为“黑衣组织”时才具有此功能。需要消耗1个毒药。

“队伍”：
同一队伍的人可以赠送道具，但是不能互相攻击。注意：在能见度很低的天气状况下可能会误伤。
进入连斗后，所有的队伍都将被取消，并且不能再组建或者加入队伍。
『创建队伍』 消耗50点体力，组建一个新的队伍，队伍必须有名称和密码。
『加入队伍』 消耗25点体力，加入一个已经存在的队伍，需要填写队伍名称和密码。每支队伍最多有5人。
『脱离队伍』 脱离目前已经加入的队伍。


“升级技能”：
可以打开技能表，使用或者学习技能。具体技能讲解见后文>>内定称号与技能点。


6.战斗说明

玩家移动或者探索时，若遇见其他的角色（包括玩家和NPC）便会发生战斗。此时，战斗的发起方是玩家自己。
战斗发起方有一定概率先制攻击。
先制的基础概率是50%，天气良好时先制率增加，天气恶劣时则降低；选用合适的策略和技能也能增加先制率。对战斗发起方而言，先制率通常是大于50%的。
如果你先制攻击，可以向对方喊话，选择或者。
如果选择攻击，则你先攻击对手，在你攻击完毕后对手再反击（也有可能无法反击）。具体的攻击方式依据你所持武器或者发动的战斗技能而有区别。而如果选择逃走，则必然回避战斗。
而如果敌方先制攻击，就是对方先发动攻击，此时直接进入战斗而无法逃跑。就算战斗发起方是你自己，也有可能被敌方先制攻击。

注意，其他玩家也会行动。当一名玩家主动行动并遭遇另一名玩家时，主动行动方就是战斗发起方，可以看到详细的作战讯息。而另一名玩家只能看到简要的战斗报告。
下面就是先制敌方时的战斗界面。左边的主界面显示双方的能力值和状态，右边则是指令区。

如果选择用武器进行攻击，则会根据自己的攻击力和熟练度，以及对方的防御力来计算攻击伤害。
如果玩家的攻击造成对方的生命值归零，那么对方就被玩家杀死了。
同样的，如果敌人的攻击造成玩家的生命值归零，那么玩家就被敌方杀害，在这局游戏中就无法再进行任何行动了。
玩家杀死对手后（无论是否是发起方），可以从对方的武器、防具、包裹道具或者金钱中，选择一项来缴获。
缴获以后无法立刻再次缴获别的道具，必须在原地再次发现尸体才有机会。


7.熟练度

武器的伤害值跟该武器类型的“熟练度”息息相关。
玩家刚进入游戏时，熟练度低下，可能用武器只能造成1-2点的伤害。
但是到后期，熟练度升高以后，可能用同样的武器都能将敌人秒杀。
战斗时，只要对敌人进行攻击，无论命中与否，武器类型对应的熟练度都会上升。
如果逃跑，自然得不到熟练度。
比较特别的是，合成物品和设置陷阱也可以增加爆系武器的熟练度。
阅读“技能书”或者使用特定的“能力药”也可以提高熟练度。
此外，使用“装有H173的注射器”进行自杀时，有微小概率会进入L5状态，此时全系熟练度都会大幅度上升。

10.道具说明

道具有以下几种类型：
『生命恢复』 使用后按道具的效果值恢复生命。
『体力恢复』 使用后按道具的效果值恢复体力。
『命体恢复』 使用后同时恢复生命和体力。
小提示：所有的恢复类道具，都有可能含毒。如果使用了带毒的恢复类道具，不仅不能恢复生命或者体力，还会损失一定的生命，并且进入“中毒”状态。

『药剂』 根据药剂类型的不同，使用后能够去除异常状态。如果不在异常状态，服用药剂完全无效。
『技能书籍』 使用后，能够按一定比例增加玩家的熟练度。如果玩家熟练度较高，技能书的效果会降低，甚至倒扣熟练度。
『强化药物』 使用后，能够按一定比例增加玩家的各项能力。如果玩家各项能力较高，能力药的效果会降低，甚至倒扣能力值。

『武器』 使用后装备在手上，增加攻击力。武器分为殴，斩，射，投，爆，灵六系，不同的武器有不同的特性，详情见下文。
『防具』 使用后装备在身体上，增加防御力，并保护该部位不会受伤。分为体，头，腕，足四个部位。
『饰品』 使用后佩戴在身体上，部分饰物有特殊用途。

『陷阱』 使用后会在当前地区设下陷阱。

『天气控制』 使用后将改变当前天气，可改变当前天气。“驱云弹”会将天气变为『大晴』，“造雾设备”会将天气变为『起雾』或『浓雾』，其他天气道具也有各自的效果。
但当天气处于“冰雹”、“龙卷风”、“瘴气”、“暴风雪”、“极光”时，无法使用天气设备改变天气。

『电脑设备』 使用后有一定概率能够解封当前禁区，让玩家可以进入禁区探索，但这一危险行动有一定概率导致该设备毁坏，甚至杀死玩家！
电脑设备使用后会消耗电力（效果值），电力降到0则需要用『电池』充电才能继续使用。

『探测仪器』 道具名一般是“生命探测器”或者“广域生命探测器”。前者使用后可以看到玩家所在地区的其他玩家以及各类NPC的数目，而后者的视野更是扩展到全地图。
无论是PVP还是PVE，掌握对手动向都是至关重要的，因而寻找/合成广域生命探测器是高水平玩家必做的事。
探测仪器使用后会消耗电力（效果值），电力降到0则需要用『探测器电池』充电才能继续使用。

『弹药』 使用后可为手中装备的射系武器，即『远程兵器』和『重型枪械』补充弹药。
分为『手枪弹药』、 『机枪弹药』、 『能源弹药』、 『气体弹药』和 『重型弹药』五种， 分别对应不同类型和属性的枪械。将鼠标悬浮在它们上可以看到详细介绍。
射系武器用尽弹药后如果继续攻击，则会当做钝器使用，不仅属性无效，攻击力还会迅速下降，因此使用射系武器时请保证弹药充足！

『合成专用』 用来合成更高级道具的物品，单独携带是没有任何作用的。
『特殊』 有特殊用途的道具，详情见“特殊道具”部分

殴、斩系武器，命中对手之后，有一定机率造成武器“损坏”。
如果武器的耐久值不是“无限”，那么耐久将会下降。耐久降为0后，武器会损毁，之后就必须赤手空拳了。
如果武器的耐久值是“无限”，那么武器的攻击效果将会下降，效果降为0时武器同样会损毁。
无限耐久的殴、斩系武器比有限耐久的殴、斩系武器更容易损坏，请注意取舍。

射系武器有“子弹”的设定，当武器耐久非“无限”时才能作为枪械进行射击。
如果射系武器子弹用尽，在游戏中耐久值将显示为“无限”。此时射系武器只能直接作为钝器攻击，攻击力会飞速下降。
根据射系武器所拥有的属性的不同，需要的子弹类型也不同，请注意收集。
带“电击”或者“音波”属性的射系武器，需要『能源弹药』；
带“火焰”或者“冰冻”属性的射系武器，需要『气体弹药』；
除此之外，连续攻击的射系武器属于“机枪”，需要『机枪弹药』；
不管带有何种属性，只要类别属于“重型枪械”，就需要『重型弹药』。
这些类型之外的射系武器属于手枪，用『手枪弹药』就可以装填。
此外，还存在属性为“一发”的枪械，这种枪械不能装填弹药，弹药用尽就永远无法作为枪械攻击。
根据子弹类型的不同，弹夹上限也不同。
装填子弹的方法是装备射系武器并「使用」子弹。

投系，爆系和灵系的武器，属于消耗品，每攻击一次数量都会减少。当然也存在可以无限攻击的这三类武器。

部分道具如“磨刀石”、 “钉”、 “武器师安雅的奖赏”等，都能强化武器，使用方法是装备武器然后使用特殊道具即可。具体情况见“特殊道具”。

13.重击和必杀

当怒气达到30时，攻击时可能发出重击，攻击伤害x1.5。
如果内定称号是“超能力者”，则可以自由控制自己的怒气，释放出“必杀技”，此时攻击伤害x2。
释放方式为在攻击时向对方喊话 。
释放必杀技会消耗50点怒气，如果不喊话则绝对不会释放必杀技。
此外，如果怒气值积累到255，则会100%发出重击或者释放必杀技。
NPC也会发出重击和必杀技，请随时小心。


14.受伤与异常状态

战斗中，玩家自己或者敌人都有一定几率发出“致伤攻击”。
致伤攻击会随机选择被攻击者的一个身体部位。
如果该部位装备有防具，则会减少该防具的耐久。如果防具耐久下降到0，那么防具将会损坏。
如果该部位未装备防具，那么被攻击者的那个部位则会“受伤”。
此外，部分武器带有“属性攻击”，有一定几率导致异常状态。

15.毒药和陷阱

陷阱类道具：「使用」之后可以在所在的区域设下陷阱，设置的同时可以增加经验值和爆系熟练度。陷阱也可能被其他玩家回避。如果未触发陷阱，玩家还能拾取该陷阱。
毒药：「使用」后可对大部份回复物品下毒。
自己设的陷阱可能会被自己触发，自己下毒的食物可能会被自己捡到并且误食，所以在设下陷阱或是下毒的时候要谨慎。
陷阱类道具会造成不超过道具效果值的伤害，下毒的食物会造成与食物效果值一样的伤害。


16.内定称号与技能点


玩家进入游戏时，会随机分配到三个内定称号，玩家可以从中任选一个，但一旦选择便无法更改。
内定称号分为“一般”，“特殊”以及“道具”三种类型。
一般内定称号增加熟练度，获得几率较高；特殊内定称号会赋予玩家特殊能力，获得几率较低。
道具类型的称号需要使用特殊的道具，或完成特殊的步骤后才能获得，是改变了游戏根本机制的上级者向玩法。

此外，玩家角色每升一级，便可以获得一点技能点。

技能点可以用于多种升级操作，其中有一些是公共的（任意称号均可使用），而有一些是特殊的（只有特定内定称号的才可使用，也只对对应称号有效）。

22.游戏周边


为了增加游戏趣味，ACFUN大逃杀还包含一些不影响游戏本身的游戏周边内容，包括成就系统、称号系统、切糕系统、玩家排行系统等。

成就系统
登陆后点击“帐号资料”导航条，进入帐号资料页面即可查看目前的所有成就和它们的完成情况。每个成就都有其要求说明，只需达到说明要求即可完成成就； 但也有几个成就是没有说明的，只能根据其标题来猜测其要求了。
完成成就可以得到一定积分和切糕奖励，有的成就还能解锁新的称号。
“当前幸存”“历史优胜”和“玩家排行”页面里的所有玩家名字都是超链接，点击它们就可以看到其他人完成成就的情况了。

称号系统
每个新的“称号”都可以通过完成特定的“成就”解锁。 当你参与游戏时，你的称号会与你的ID一起出现在游戏进行状况中。
登陆后点击“帐号资料”导航条，然后点击“编辑我的帐号资料”按钮，即可进入编辑帐号资料页面。 选择你喜欢的称号后提交即可。选择的新称号在下一局游戏时才会生效。

切糕系统
你可以在“当前幸存”页面中，选择你认为将获得游戏胜利的人，使用“切糕”对其下注。
下注后，切糕进入当局游戏奖池，当游戏结束时，判断正确者将以下注多少和分成系数为依据，分得奖池中的切糕。该局的优胜者也可以获得一定比例的切糕奖励。
仅当游戏不处于“连斗”状态且处于0禁时，才可以进行下注。
因为下注时间越早，判断难度越大，风险也越高，因此我们将使用“分成系数”概念。
当游戏开始时，有一个初始的分成系数，这个分成系数会随着游戏的进行而逐渐减少。 你的实际分成权值将是你的下注金额乘以下注时的分成系数。 这样在较早时刻下注时，同等切糕将有更大的分成权值，从而做到相对公平。
每人每局只能最多下注1000切糕。 每局有固定的奖池上限，为8000切糕。 当当前奖池达到奖池上限后，玩家依然可以下注，但每人只能下注不超过50切糕。 这是为了防止奖池过大，保证每局游戏不会使切糕总数增加过多。
每次参加游戏均会奖励10切糕（游戏结束时发放），完成成就也可以获得切糕奖励，获得游戏的胜利也可以从这局游戏的总奖池中分到一些切糕。

此外，你还可以为指定玩家投资，使用“切糕”购买“商店道具”并召唤一位快递员进入战场为其派送。
但是你为场内玩家投资的道具也有可能被其他玩家劫走。

玩家排行系统
积分表现出你参与游戏时间的多少。
每参加一局游戏，均可以获得一定战斗力积分。 获得的积分与你在该局中的表现有关，积分会累计。完成成就也可以获得积分奖励。
在“玩家排行”中可以看到积分排行榜、胜率排行榜和切糕排行榜。


23.单机版安装指南


ACFUN大逃杀是开源的网页游戏，你可以在这里或这里下载到最新的源代码。
如果你希望安装单机版：
0. 请先在本地电脑上准备好apache+mysql+php环境，将游戏代码包解压至本地服务器中。随后可执行install.php，跟随向导进行来安装游戏，也可以按照以下方式，进行手动安装。
请注意：本游戏的 Native 依赖为PHP5.4，MySQL 5.5，在高版本PHP和MySQL下安装很可能会出现问题！
1. 进入mysql
mysql -u用户名 -p密码
2. 建立并进入数据库。
create database acdts;
use acdts;
3. 完整的数据表结构在游戏目录下./gamedata/sql/all.sql中，将其导入数据库中： （将下面命令中路径改为自己机器上该文件路径）
source /srv/http/dts/gamedata/sql/all.sql
4. 向数据表中加入初始化游戏必需的记录
insert into acbra2_users (gid) values (0);
insert into acbra2_winners (gid) values (0);
insert into acbra2_game (gamenum) values (0);
5. 配置config.inc.php，将游戏目录下config.inc.php.sample改名为config.inc.php，并按其中要求填写数据库用户名、密码、超级管理员帐号等信息。
6. 用浏览器进入游戏，如果能显示出页面，没有错误信息，就说明基本成功了。 在帐号注册中注册config.inc.php中填写的超级管理员帐号，然后登录这个帐号即可进行游戏管理。
